# 基于OpenCV的人脸识别打卡/签到管理系统（可基于树莓派）

## 项目简介

本项目是一个基于OpenCV的人脸识别打卡/签到管理系统，适用于毕业设计或相关课程设计。系统支持在树莓派等嵌入式设备上运行，能够实现高效、准确的人脸识别功能，并记录用户的打卡/签到信息。

## 功能特点

- **人脸识别**：利用OpenCV库进行人脸检测和识别，确保打卡/签到的准确性。
- **打卡/签到记录**：系统能够自动记录用户的打卡/签到时间，并生成相应的记录文件。
- **树莓派支持**：系统设计考虑了树莓派等嵌入式设备的硬件特性，能够在资源受限的环境下稳定运行。
- **用户管理**：支持用户信息的录入、修改和删除，方便管理员进行用户管理。

## 使用说明

1. **环境配置**：
   - 安装Python 3.x
   - 安装OpenCV库：`pip install opencv-python`
   - 安装其他依赖库：`pip install -r requirements.txt`

2. **系统运行**：
   - 运行主程序：`python main.py`
   - 系统启动后，用户可以通过摄像头进行人脸识别打卡/签到。

3. **用户管理**：
   - 管理员可以通过系统界面进行用户信息的录入、修改和删除操作。

## 注意事项

- 系统运行需要摄像头支持，请确保设备已正确连接摄像头。
- 在树莓派等嵌入式设备上运行时，建议优化代码以提高运行效率。
